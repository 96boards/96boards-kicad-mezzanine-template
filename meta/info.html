96Boards

Mezzanine Board Template

This is a project template for developing a 96Boards Mezzanine board (compatible with these boards).

This project's schematic only contains the necessary connectors needed for a mezzanine board. It comes in two formats: Through Hole Connector, or Surface Mounted Connectors.

The PCB includes the aforementioned components, plus mounting holes. The holes and connectors have been positioned so that the boards are aligned when mounted. The edge cuts also aligns with the base board. All the mezzanine board IO is connected to the header pins.
