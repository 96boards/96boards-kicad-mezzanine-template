This is a project template for developing a 96Boards Mezzanine board (compatible with these boards).
This project's schematic contains the necessary connectors, level shifters, volatage regulator and FDI.
The PCB includes all of the mentioned components, plus mounting holes. The holes and connectors have been positioned so that the boards are aligned when mounted. The edge cuts also aligns with the base board. All the mezzanine board IO is connected to the header pins.
